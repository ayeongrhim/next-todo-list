# Do it! - Todo List

할 일 목록을 관리하는 Todo 서비스입니다.

## 기술 스택

- Next.js 15
- TypeScript
- Tailwind CSS

## 시작하기

### 설치

\```bash
npm install
\```

### 개발 서버 실행

\```bash
npm run dev
\```

브라우저에서 [http://localhost:3000](http://localhost:3000) 으로 접속하세요.

## 주요 기능

- 할 일 추가 / 삭제 / 수정
- 진행 중 / 완료 상태 관리
- 이미지 첨부 (영어 파일명, 5MB 이하)
- 메모 작성
- 반응형 웹 디자인 (모바일 / 태블릿 / 데스크탑)
